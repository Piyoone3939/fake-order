# Figma UI簡易デザイン

このFigmaファイルは、画面構成・情報量・ロール別UIを確認するための簡易デザインです。最終ビジュアルでは、AI生成風の質感を抑えて各要素を再設計します。

## 作成済み画面

1. タイトル
2. ロール選択
3. スパイ準備
4. オーガナイザー準備
5. スパイ・メインHUD
6. オーガナイザー・メイン画面
7. リザルト

補助UIとして、端末操作、ハッキング、命令ログ、罠配置のオーバーレイ例も作成しています。

## デザインシステム

- Foundations: カラー、タイポグラフィ、余白、影
- Components: ロール別ボタン、状態チップ、文書パネル、監視カメラ
- フォント: Noto Sans JP、IBM Plex Mono
- ロールカラー: スパイはターコイズ、オーガナイザーはオレンジ
- 情報状態: 警告はアンバー、危険はコーラル、偽情報はバイオレット

## 確認状況

- 7画面の未処理プレースホルダーがないことを確認済み
- 仮レイヤー名が残っていないことを確認済み
- 指定外フォントが使われていないことを確認済み
- ロールカラー上の文字コントラストを調整済み
- ロール選択、スパイHUD、オーガナイザー画面、リザルトを画像で確認済み
- 表紙とタイトル画面を正式な横長ロゴへ差し替え済み

このファイルはレイアウトと画面遷移の確認用です。Unityへの画面実装と最終デザイン制作は次の工程で行います。
