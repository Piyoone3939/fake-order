# Fake Order 操作方法

この文書は、現在の Unity ローカルプロトタイプの実装に対応した操作ガイドです。

## 起動方法

1. Unity Hub から `FakeOrder` フォルダーを Unity 6000.4.1f1 で開きます。
2. Project ウィンドウから `Assets/Scenes/GamePrototype.unity` を開きます。
3. 初回のスクリプト読み込みが終わるまで待ち、Hierarchyに `Spy`、`Organizer`、`SpyUI`、`OrganizerUI` などが表示されたことを確認します。
4. `Ctrl + S` で、配置されたオブジェクトをシーンへ保存します。
5. Unity Editor 上部の Play ボタンを押します。
6. Play モードになったら `Game` タブを選択します。

旧状態のシーンでは最初のHierarchyに `Main Camera`、`Directional Light`、`GameObject` だけが表示されます。Playモードを停止した状態でスクリプトの再読み込みが完了すると、`GameObject` の `GamePrototypeSetup` が編集用オブジェクトを一度だけ配置します。それ以降はPlay開始時に作り直されないため、通常のシーンオブジェクトとして編集できます。

## プロトタイプの概要

- 現在はスパイとオーガナイザーを同じ PC・同じキーボードで操作するローカル検証版です。
- 開始後は60秒間の準備フェーズがあり、その後メインフェーズへ移行します。
- スパイは異なる3台の端末から新鮮な本物の情報を収集し、脱出ポイントから脱出すると勝利です。
- 情報が1～2件の状態で脱出すると部分成功、情報なしで脱出すると失敗になります。
- メインフェーズの制限時間は15分です。

## ロール選択

Playすると最初にロール選択画面が表示されます。

| ロール | 操作 |
|---|---|
| スパイを選択 | 画面の「スパイ」をクリック、または `1` |
| オーガナイザーを選択 | 画面の「オーガナイザー」をクリック、または `2` |
| ロール選択へ戻る | ゲーム中に `F1` |

選択後は60秒の準備フェーズが始まり、画面上部に残り時間が表示されます。メインフェーズが始まると、選択したロールの入力だけが有効になります。
準備を省略してすぐ確認したい場合は、準備フェーズ中に `Enter` を押します。

## スパイ側の操作

| 操作 | キー／入力 |
|---|---|
| 移動 | `W` `A` `S` `D` または矢印キー |
| 視点移動 | マウス移動 |
| 走る | 移動中に左 `Shift` |
| 端末・脱出ポイントを操作 | 対象を見ながら `E` |
| マウスカーソルのロック／解除 | `Esc` |

走り始めると疑惑ゲージが上昇します。操作可能な対象を見ると、画面下部に `[E]` のプロンプトが表示されます。

### 端末操作

端末を見ながら `E` を押し、表示されたメニューから次の操作を選びます。

| 操作 | キー |
|---|---|
| 情報収集 | `1` |
| 命令書偽造：情報漏洩 | `2` |
| 命令書偽造：警備移動命令 | `3` |
| 命令書偽造：点検命令 | `4` |
| 命令書偽造：緊急命令 | `5` |
| メニューをキャンセル | `E` |

- 選択後、ハッキング完了まで約5秒かかります。
- ハッキング中またはメニュー表示中に端末から離れすぎると、自動的にキャンセルされます。
- 同じ端末を再度収集すると情報の鮮度は更新されますが、収集件数は増えません。
- 収集した情報は画面上部に表示され、時間経過に応じて鮮度が変化します。
- 罠情報は取得時点では本物と同じ表示になるため、UIの色だけでは判別できません。
- 収集済み端末はミニマップに追加されます。

### 脱出

施設内の `ExitPoint` を見ながら `E` を押します。脱出結果は、収集した「新鮮かつ罠ではない情報」の数によって決まります。

| 情報数 | 結果 |
|---|---|
| 3件以上 | スパイ勝利 |
| 1～2件 | 部分成功 |
| 0件 | 失敗 |

## オーガナイザー側の操作

| 操作 | キー／入力 |
|---|---|
| 命令ログを開く／閉じる | `L` |
| 監視カメラの強調表示を切り替える | `C` |
| 罠情報の配置メニューを開く／閉じる | `T` |
| 俯瞰マップを拡大 | マウスホイール上 |
| 俯瞰マップを縮小 | マウスホイール下 |

監視カメラにはエリア名、設定された遅延時間、映像時刻が表示されます。`C` を押すたびに強調対象が切り替わります。

### 命令ログ

`L` で命令ログを開いた後、次のキーで操作します。

| 操作 | キー |
|---|---|
| 上のログを選択 | `I` |
| 下のログを選択 | `K` |
| 選択した命令を無効化 | `Enter` またはテンキー `Enter` を5秒間長押し |

- 無効化の長押し中は進捗バーが表示されます。途中でキーを離すと進捗はリセットされます。
- 命令を無効化すると、そのエリアは5分間、偽造命令を受け付けなくなります。
- オーガナイザーが命令ログを開いている間、スパイ側には「命令ログを確認中」と通知されます。
- 同一エリアで偽造命令が3回発行されるごとに、オーガナイザー側へ警告が表示されます。

### 罠情報の配置

罠情報は準備フェーズ中またはメインフェーズ中に配置できます。

1. `T` で罠情報配置メニューを開きます。
2. `1`、`2`、`3` のいずれかで対象端末を選択します。
3. そのまま10秒待つと配置が完了します。

| 操作 | キー |
|---|---|
| 端末 #1を選択 | `1` |
| 端末 #2を選択 | `2` |
| 端末 #3を選択 | `3` |
| メニューを閉じる／配置をキャンセル | `T` または `Esc` |

- 配置中は進捗バーが表示されます。
- 配置済みの端末は選択メニューに表示されます。
- 配置を完了した端末からスパイが情報収集すると、罠情報を取得します。
- 罠情報はスパイ側では本物と同じ色で表示されます。

## 現在の注意点

- 現在は1台のPCで片方のロールを操作する検証版です。選択していない側は自動操作されません。
- `Esc` は端末メニューのキャンセルではなく、マウスカーソルのロック切り替えです。端末メニューは `E` でキャンセルします。
- 現在のシーンはコアシステム確認用の簡易施設です。オンライン対戦とNPCはまだ実装されていません。
